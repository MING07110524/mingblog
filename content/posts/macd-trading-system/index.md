## 模式概述

**MACD趋势回踩买入模式**，核心逻辑：不在死叉时硬做，只做上涨趋势里的回踩二次启动。

| 要素 | 说明 |
|------|------|
| 适用环境 | 大盘不处于单边下跌；个股属热点板块或中期趋势向上 |
| 持股周期 | 3 天到 3 周（日线波段） |
| 核心工具 | MACD（趋势+动能）、均线（位置）、成交量（确认真假） |

---

## 买入条件

1. 股价站上 20 日线，20 日线走平或向上
2. MACD 在 0 轴上方，或**接近 0 轴后再次金叉**
3. 前面已经有一波上涨，回踩时**缩量**
4. 再启动当天**放量阳线**，最好突破回踩平台
5. 优先做热点板块前排，不做冷门杂毛

### 两点买法

- **第一买点**：MACD 二次金叉当天，放量突破
- **第二买点**：突破后次日回踩不破关键位，低吸
- **禁忌**：不在大阳线拉高 5% 以上时追涨

---

## 卖出与止损

### 卖出条件

| 信号 | 操作 |
|------|------|
| MACD 高位死叉 | 先减仓 |
| 跌破 10 日线或 20 日线且收不回 | 离场 |
| 放巨量但股价涨不动 | 分批止盈 |
| 短期盈利 8%~15% | 先落袋一部分 |

### 止损规则

- 短线票：买入后亏 **3%~5%** 止损
- 若买入逻辑失效（如放量突破失败），**当天或次日就走**

---

## 两种核心形态

### 最强形态（主做对象）

- 股价站稳 20 日线
- DIF、DEA 位于 0 轴上方
- 回踩时 MACD 绿柱缩短
- 再启动时红柱重新放大

### 弱转强形态（只能轻仓试错）

- 前期下跌结束后，MACD 在 0 轴下方金叉
- 随后股价重新站上 20 日线
- 回踩不破后再次转强

## 仓位规则

| 阶段 | 仓位 |
|------|------|
| 首次试仓 | 总仓 20%~30% |
| 突破确认后加仓 | 再加 20%~30% |
| 大盘弱/情绪弱 | 降仓处理 |

## 选股流程

1. **先选强板块**：只做近期成交活跃、联动明显的方向
2. **板块内筛选**：站上 20 日线 + 回踩缩量 + MACD 即将金叉
3. **只在放量突破时买**：不提前埋伏赌突破

---

## 通达信选股公式

### 趋势筛选
```text
C > MA(C,20);
MA(C,20) >= REF(MA(C,20),1);
```

### MACD筛选
```text
DIFF:=EMA(C,12)-EMA(C,26);
DEA:=EMA(DIFF,9);
MACD金叉: CROSS(DIFF,DEA);
0轴上方: DIFF > 0 AND DEA > 0;
```

### 回踩后启动筛选
```text
缩量回踩: V < MA(V,5) AND C / HHV(C,10) > 0.92;
放量启动: V > MA(V,5) * 1.2 AND C > REF(H,1);
```

### 组合选股
```text
选股: C > MA(C,20)
AND MA(C,20) >= REF(MA(C,20),1)
AND CROSS(EMA(C,12)-EMA(C,26),EMA(EMA(C,12)-EMA(C,26),9))
AND EMA(C,12)-EMA(C,26) > 0
AND V > MA(V,5) * 1.2;
```

## 执行口诀

> 先看板块，再看趋势，再等回踩，再等金叉放量，破位就走。

## 禁做清单

- 禁止在下降趋势里仅凭金叉抄底
- 禁止在大阳线拉升 5% 以上后情绪化追高
- 禁止在板块退潮时逆势做后排股
- 禁止无止损计划开仓
